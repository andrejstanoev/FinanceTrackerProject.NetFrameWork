# FinanceTrackerProject
Оваа веб-апликација е изработена со .NET Framework и претставува систем за следење на лични финансии. Проектот е инспириран од мој претходен проект — исто така “Finance Tracker” — кој беше изграден со React и Firebase. Овојпат, одлучив да создадам целосно серверска апликација користејќи го .NET Framework-от за да го подобрам моето познавање на back-end развојот и MVC архитектурата.

## Опис на апликацијата
Со оваа апликација корисниците можат:
- Да се регистрираат или најават доколку веќе имаат корисничка сметка.
- По успешна најава се пренасочуваат кон Home Page, каде што се прикажани нивните лични информации како:
    - Име
    - Е-пошта
    - Број на креирани буџети
    - Број на трошоци
    - Вкупен износ од сите буџети (во долари)
На почетната страница, корисникот може да ги види и своите три последно креирани буџети. Ако има помалку од три, ќе бидат прикажани само постоечките.

## Функционалности
Апликацијата овозможува:
 - Создавање, уредување и бришење на буџети
 - Создавање, уредување и бришење на трошоци
 - Во деталниот приказ на секој буџет, може да се создадат, уредат или избришат трошоци
 - Преглед на сите буџети и трошоци преку интуитивен и едноставен интерфејс

## Структура на апликацијата
Апликацијата се состои од три главни страници:
 1. Home Page – прикажува лични податоци и преглед на последни буџети
 2. Budgets Page – управување со буџети (CRUD операции)
 3. Expenses Page – управување со трошоци (CRUD операции и сортирање според некој атрибут)

## Модели
Во проектот се користат три основни модели:
- User Model – ги содржи податоците за корисникот (име, е-пошта, лозинка и слично)
- Budget Model – ги содржи податоците за буџетите (наслов, износ, датум на креирање...)
- Expense Model – ги содржи податоците за трошоците (опис, износ, датум, поврзан буџет)
Дополнително, се користат View Models кои овозможуваат пренесување на податоци помеѓу контролерите и погледите и обратно.

## Технологии и архитектура
- .NET Framework (.NET MVC архитектура)
- Entity Framework за работа со базата на податоци
- C# како главен програмски јазик
- HTML, CSS и Bootstrap за фронт-енд изгледот

## Цел и инспирација
Целта на овој проект е развивање на стабилна и функцонална веб-апликација за управување со лични финансии. Проектот произлезе како идеја од претходен React + Firebase проект, но овојпат со фокус на серверска логика и база на податоци преку .NET Framework.
